Esse trecho:

```python
(
    df['Livre/Guiada']
    .value_counts(normalize=True)
    .mul(100)
    .round(2)
```

# Isso fica onde no dashboard?

Esse tipo de cálculo normalmente vira:

* KPI
* Pie chart
* Insight textual

---

# 1. Melhor uso → gráfico Pizza (Pie)

Porque mostra divisão de participação.

Exemplo com Plotly:

```python
base = (
    df['Livre/Guiada']
    .value_counts(normalize=True)
    .mul(100)
    .round(2)
    .reset_index()
)

base.columns = ['tipo_visita', 'percentual']

fig = px.pie(
    base,
    names='tipo_visita',
    values='percentual',
    title='Distribuição de Visitas'
)

st.plotly_chart(fig, use_container_width=True)
```

---

# 2. Pode virar KPI também

Exemplo:

```python
perc_guiada = (
    df['Livre/Guiada']
    .value_counts(normalize=True)
    .mul(100)
    .round(2)
    .get('Guiada', 0)
)
```

Depois:

```python
st.metric(
    'Visitas Guiadas (%)',
    f'{perc_guiada}%'
)
```

---

# 3. Também gera insight automático

Exemplo:

```python
A maior parte das visitas foi do tipo Livre,
representando 72.45% do total.
```

Isso fica muito forte no portfólio porque mostra:

* análise de comportamento
* leitura de negócio
* storytelling com dados
